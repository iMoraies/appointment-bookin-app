# Booking App (Frontend Project)

A simple booking application built with HTML, CSS and Vanilla JavaScript.
This project demonstrates frontend fundamentals, clean structure and basic state management.

---

## Features

- Service selection
- Date and time booking
- Booking summary
- LocalStorage persistence
- Clean HTML structure
- Vanilla JavaScript (no frameworks)

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

---

## Project Structure
/
├── index.html
├── README.md
├── css/
│ └── styles.css
└── js/
└── app.js


## How to Run

Just open `index.html` in your browser.
